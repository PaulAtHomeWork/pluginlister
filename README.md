![Plugin Lister screenshot](https://github.com/PaulAtHomeWork/pluginlister/blob/main/Plugin_lister_1.jpg)



## 👋 Welcome

Thank you for downloading **PluginLister**!

This is version **1.0**, intentionally designed to be **simple and minimal**.

This tool may be useful if:

- You install many plugins 
- You need to share your plugin list with a studio (including versions for compatibility) 
- You want to keep track of installed plugins and their versions 

------

## 🚀 How to Use

No installation required — the software is **portable**.

1. Extract the ZIP file into a folder (for example, on your Desktop) 
2. Open the folder 
3. Double-click **PluginLister.App.exe** (the one with the icon) 
4. Select the folder where your VST3 plugins are installed 
5. Wait a few seconds (or minutes, depending on the number of plugins) 
6. The plugin list will appear 

------

## 📦 Features

- Lists installed plugins with version and path 
- Portable (can run from any folder or USB drive) 
- Free to use 
- Saves preferences 
- Export plugin list to **PDF** 

------

## 💻 Requirements

- Windows 11 (should also work on Windows 10) 
- [.NET 8.0 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) 

👉 Try running the app first — you may already have the runtime installed.
